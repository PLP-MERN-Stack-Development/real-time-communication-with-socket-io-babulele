## How to Take Screenshots

### Desktop Screenshots:
1. Start the application (server and client)
2. Open the application in your browser
3. Use browser screenshot tools or OS screenshot tools
4. Save screenshots in PNG format
5. Recommended resolution: 1280x720 or higher

### Mobile Screenshots:
1. Use browser dev tools (F12) to switch to mobile view
2. Or use actual mobile device and take screenshots
3. Save screenshots in PNG format
4. Recommended resolution: 375x667 (iPhone) or 360x640 (Android)

### Tips:
- Take screenshots in different scenarios (multiple users, different rooms, etc.)
- Show key features clearly
- Use descriptive file names
- Keep file sizes reasonable (< 1MB per image)

## File Naming Convention

Use kebab-case for file names:
- `login-screen.png`
- `main-chat.png`
- `private-chat.png`
- etc.

## Formats Supported

- PNG (recommended for screenshots)
- JPG (acceptable)
- GIF (for animated demonstrations)
